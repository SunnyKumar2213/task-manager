# task-manager
Complete backed for Task Manager app with JWToken verification.  
Create User Create a task for that user Create a user profile of that user Create/update the user profile.
Add experience of the user comments, Likes, publish the post delete user, delete all users, delete task when the user is removed. 
Requirements:  Lastest Node js in your system npm install. 
How to run the project? open CMD /Terminal in your system  cd task-manager npm install cd src node index.js
